  2. defaultdict 是 Python 标准库 collections 模块中的一个类，它是内置 dict 类的子类，主要用于简化字典中缺失键的处理。

与普通字典不同，defaultdict 在初始化时需要指定一个默认工厂函数（如 list、int、set 等），当访问字典中不存在的键时，会自动调用该工厂函数创建一个默认值，而不是抛出 KeyError 异常。

类似 if key not in d: d[key] = []
